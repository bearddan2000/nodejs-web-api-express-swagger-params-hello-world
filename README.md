# nodejs-web-api-express-swagger-params-hello-world

## Description
Shows REST api documentation through swagger.
Passes a parameter through the url.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
- http://localhost/api-docs
- http://localhost/api/v1

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
